# art-marketplace-microservice

The microservice application enables artists to sell art, and buyers to discover and buy art.


## Services:
- User service
- Art Service
- Order Service
- Payment Service
- Notification Service(Email)

## Technology Tool:
- FastAPI
- Next JS(SSR with hydration for dynamic interactions post server side rendering)
- NGINX for load balancing
- AWS S3 for Image Upload
- Postgresql
- Redis
- Docker
- Kubernetes(For practice)
- Maybe Monitoring
