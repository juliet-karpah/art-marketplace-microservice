# ai-art-marketplace-microservice

The microservice application enables artists to sell art, and buyers to discover and buy art. AI is used to estimate price, generate tags, and descriptions for every listing, and AI-powered semantic search and recommendations based on previous activities. 

This is such a fun project as it incorporates microservices and AI.

## Services:
- User service
- Art Service
- Recommendation Service
- Order Service
- Payment Service
- Notification Service(Email)

## Technology Tool:
- FastAPI
- Next JS(SSR with hydration for dynamic interactions post server side rendering)
- NGINX for load balancing
- AWS S3 for Image Upload
- Vector Database for embedded 
- Postgresql
- Redis
- Docker
- Kubernetes(For practice)
- Maybe Monitoring
